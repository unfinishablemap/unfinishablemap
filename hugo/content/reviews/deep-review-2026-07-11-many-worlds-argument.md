## The Finding (verified against the live file + apex)

Argument 3 indicted MWI because the decoherence-preferred basis "depends on the environment and is not unique," concluding "the branches we experience may be artifacts of how we decompose the wave function rather than objective features of reality." But the Map's **own** preferred mechanism — post-decoherence selection — requires consciousness to select which element of the already-decohered mixture becomes actual, and that mixture **is** the decoherence-preferred pointer basis.

Cross-checked against the apex [post-decoherence-selection-programme](/apex/post-decoherence-selection-programme/): it explicitly has Zurek's quantum-Darwinism **einselection** "specify the menu" of definite outcomes ("The menu is fixed. Einselection determines which states survive environmental monitoring"), while itself conceding (line 97) that "quantum Darwinism's core results — einselection, redundant encoding, the emergence of objectivity — are interpretation-neutral. They work equally well in collapse interpretations ... and consciousness-mediated frameworks." So the same einselection resource the Map claims hands *it* a well-defined menu is equally available to Wallace's Everettian, who individuates branches by exactly the same decoherence structure. The disjunction bites: **either** einselection fixes the basis (then Argument 3 fails as an *independent* objection and the real disagreement is collapse-realism, i.e. Argument 1) **or** it does not (then the Map's own post-decoherence menu is as observer-relative as MWI's branches). The article wielded the objection one-directionally.

This is a genuine internal-coherence defect (a bidirectional-consistency / self-application issue), **not** a bedrock framework-boundary disagreement — a Tenet-accepting reviewer would flag it, which is the diagnostic test for correctability. It does not touch the No-Many-Worlds commitment: the genuine, independent anti-MWI advantage (real collapse in a definite basis, which MWI declines to posit) is *relocated*, not abandoned.
